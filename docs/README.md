# VietRecruit Documentation

This directory centralizes all technical documentation for the VietRecruit application, structured according to the Diátaxis framework. 

## Directory Index

*   **[adr/](./adr/)**: Architecture Decision Records detailing historical technical choices and their rationale.
*   **[api/](./api/)**: API references, request/response formats, and error handling schemas.
*   **[architecture/](./architecture/)**: High-level system design, infrastructure topology, and C4 architectural models.
*   **[features/](./features/)**: Domain module specifications, business logic constraints, and feature-specific implementation details.
*   **[flows/](./flows/)**: Sequential state diagrams and procedural documentation for core application flows (e.g., Applicant Tracking, Authentication).
*   **[future/](./future/)**: Roadmap features and planned architectural expansions (e.g., AI Matching, Onboarding).
*   **[runbooks/](./runbooks/)**: Operational guides, local development setup instructions, and troubleshooting procedures.

## Contribution Standards

Read [DOCS.md](./DOCS.md) prior to modifying or generating documentation. All documentation updates must strictly adhere to the guidelines specified within that file.
